# Spec: 006-brand-site

## Feature Name
광고주 의뢰 사이트

## User Value (고객 가치)
- **쉬운 광고 의뢰**: 복잡한 협상 없이 3분 만에 캠페인 등록
- **타겟팅 자동화**: 제품 카테고리 선택하면 최적 페르소나 자동 추천
- **투명한 가격**: 명확한 요금제로 예산 계획 용이
- **빠른 실행**: 승인 즉시 캠페인 시작 (기존 협찬 2주 → 2일)

## Target User (누가 쓸까?)
- **Primary**: 브랜드 마케터 (소규모 브랜드, 스타트업)
- **Use Case**:
  - 신제품 론칭 캠페인
  - 시즌 프로모션 (여름 세일, 크리스마스)
  - 브랜드 인지도 확대

## User Journey & Screen Flow

### 1. 광고주 랜딩 페이지
- **UI Elements**:
  - 서비스 소개 (100명의 AI 인플루언서 네트워크)
  - 핵심 가치 3가지 (저렴함, 빠름, 다양성)
  - 가격표 (패키지별)
  - 포트폴리오 (기존 광고 사례)
  - CTA: "캠페인 시작하기"
- **CTA**: "캠페인 시작하기"
- **Next**: 브랜드 회원가입 또는 로그인

### 2. 브랜드/상품 등록
- **UI Elements**:
  - 브랜드명, 로고 업로드
  - 제품 정보 (이름, 카테고리, 이미지, 설명)
  - 제품 URL (구매 링크)
  - 제품 카테고리 선택 (Beauty, Fashion, Food, Fitness, etc.)
- **CTA**: "다음 단계"
- **Next**: 타겟 선택

### 3. 타겟 페르소나 선택
- **UI Elements**:
  - 선택한 카테고리 기반 페르소나 추천 목록
  - 각 페르소나 카드: 프로필 이미지, 이름, 카테고리, 예상 도달률
  - 필터: 세부 카테고리, 연령대, 스타일
  - 선택 개수에 따른 예상 비용 계산기
- **CTA**: "페르소나 선택 (최소 5명, 최대 50명)"
- **Next**: 캠페인 기간 설정

### 4. 캠페인 기간 & 예산 설정
- **UI Elements**:
  - 시작일/종료일 캘린더
  - 캠페인 기간 (1주/2주/1개월)
  - 포스팅 빈도 (페르소나당 1-3회)
  - 총 예상 비용 계산 (페르소나 수 × 포스팅 횟수 × 단가)
  - 결제 정보 입력
- **CTA**: "결제 및 캠페인 시작"
- **Next**: 결제 처리 → 캠페인 승인 대기

### 5. 캠페인 대시보드 (광고주용)
- **UI Elements**:
  - 진행 중 캠페인 목록
  - 각 캠페인: 선택 페르소나, 포스팅 현황, 도달률, 참여율
  - 콘텐츠 미리보기 (007-ad-content에서 생성된)
  - 승인 대기 중인 광고 콘텐츠 (승인/거절 가능)
- **CTA**: "새 캠페인 시작", "성과 리포트 보기"
- **Next**: 010-analytics (성과 분석)

## Business Rules

### 가격 정책
- **단가 기준**: 페르소나당 1회 포스팅 = 5만원
- **패키지 할인**:
  - Basic (5-10명, 1주): 10% 할인
  - Standard (11-30명, 2주): 20% 할인
  - Premium (31-50명, 1개월): 30% 할인
- **추가 옵션**:
  - 스토리 포스팅 추가: +20%
  - 특정 시간대 지정: +10%

### 페르소나 매칭 규칙
- **자동 추천 로직**:
  - 제품 카테고리 = 페르소나 카테고리 일치
  - 예상 도달률 높은 순 정렬
- **수동 선택 가능**: 추천 외 페르소나도 선택 가능

### 캠페인 승인 프로세스
- **자동 승인**: 정상 결제 완료 시
- **관리자 검토**: 부적절한 제품/브랜드 필터링
- **승인 소요 시간**: 최대 24시간

### 광고 콘텐츠 승인
- 광고주는 007-ad-content에서 생성된 광고 콘텐츠 사전 승인 가능
- 승인 기한: 24시간 (미승인 시 자동 승인)

## Edge Cases

### 결제 실패
- 카드 승인 실패: 다른 결제 수단 요청
- 예산 초과: 페르소나 수 또는 기간 조정 제안

### 부적절한 제품
- 성인용품, 도박, 불법 제품: 자동 거절
- 의약품, 건강기능식품: 관리자 수동 검토

### 캠페인 중도 취소
- 시작 전 취소: 100% 환불
- 진행 중 취소: 남은 기간 비율만큼 환불
- 완료 후 취소: 환불 불가

## Dependencies
- **001-persona**: 광고주에게 페르소나 목록 제공
- **연관 기능**:
  - 007-ad-content: 캠페인 정보 기반 광고 콘텐츠 생성
  - 010-analytics: 캠페인 성과 리포트 제공
