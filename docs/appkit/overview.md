# AI Influencer Network - 서비스 컨셉

## 🎯 서비스 본질
"검증된 인플루언서 스타일을 조합해 광고주에게 제공하는 AI 인플루언서 네트워크"

실제 성공한 인플루언서 콘텐츠 요소를 크롤링하고, 관리자가 장소+제품+포즈를 조합해 새로운 콘텐츠를 생성. 100명의 AI 인플루언서가 다양한 카테고리에서 브랜드 광고를 자연스럽게 노출.

## 🔥 해결하는 문제

**광고주(브랜드)가 겪는 문제**:
- 우리 제품에 맞는 인플루언서 찾기 어려움
- 소규모 브랜드는 인플루언서 협찬 비용 부담 (1회 100만원+)
- 다양한 타겟층에 동시 노출하려면 여러 인플루언서 섭외/관리 필요
- 콘텐츠 퀄리티 불확실, 시행착오 반복

**우리의 해결책**:
- 브랜드 카테고리별 최적화된 AI 인플루언서 100명 보유
- 검증된 콘텐츠 요소 조합 (이미 성공한 레퍼런스 계정 기반)
- 빠른 제작: 크롤링 결과에서 선택 → 나노바나나로 자동 생성
- 자연스러운 캡션 자동 생성 (레퍼런스 톤앤매너 유지)

## 👥 타겟 고객

### Primary Persona
- 김대표 (35세, 뷰티 브랜드 운영)
- Pain: "신제품 론칭인데 인플루언서 섭외 비용 500만원은 부담돼요. 그리고 1명만으로는 다양한 타겟층 공략이 어려워요"
- Want: 저렴한 비용으로 다양한 스타일의 인플루언서 마케팅

### Secondary Personas
- 스타트업 마케터: 빠른 캠페인 실행, 다양한 A/B 테스트
- 마케팅 에이전시: 클라이언트 캠페인 대행용 인플루언서 풀 확보
- 공동구매 플랫폼: 다양한 인플루언서 채널로 노출 확대

## 💎 핵심 가치 제안

1. **검증된 요소 조합**: 이미 성공한 인플루언서 콘텐츠 요소 재조합
2. **빠른 제작**: 크롤링 결과 선택 → 이미지 자동 생성 → 캡션 자동 작성
3. **자연스러운 톤**: 레퍼런스 계정 스타일 유지한 캡션
4. **다양한 연출**: 100가지 다른 스타일로 동일 상품 노출
5. **24/7 자동 소통**: DM/댓글 자동 응답으로 팔로워 관리

## 📦 주요 기능 (고객 가치 중심)

### Phase 1: 캐릭터 셋업 (초기 1회)

**001-persona**: 100명의 다양한 인플루언서 생성
- 카테고리별 페르소나 자동 생성
- 미드저니용 프롬프트 export
- 프로필 이미지 업로드 시스템

**002-reference-crawl**: 성공 사례 크롤링으로 검증된 스타일 확보
- 레퍼런스 인스타 계정 크롤링
- 포스트 이미지, 캡션, 해시태그 수집
- 캡션 톤앤매너 자동 분석

### Phase 2: 콘텐츠 제작

**003-content-composer**: 크롤링 요소 조합으로 빠른 제작 (핵심!)
- 크롤링 결과 갤러리 뷰
- 장소/제품/포즈/분위기 선택 조합
- 나노바나나 프롬프트 자동 생성
- 이미지 즉시 생성 & 미리보기

**004-caption-generator**: 자연스러운 캡션 자동 작성
- 생성 이미지 맥락 분석
- 레퍼런스 톤앤매너로 캡션 생성
- 해시태그 자동 추천
- 수정 가능한 에디터

**005-approval**: 퀄리티 관리
- 이미지+캡션 최종 확인
- 승인/수정/거절
- 포스팅 스케줄 설정

### Phase 3: 광고 캠페인

**006-brand-site**: 광고주가 쉽게 의뢰
- 브랜드/상품 등록
- 타겟 카테고리 선택 → 적합 캐릭터 자동 추천
- 캠페인 기간 설정

**007-ad-content**: 광고도 자연스럽게
- 상품을 콘텐츠에 자연스럽게 합성
- 광고 문구 캡션 자동 삽입
- 승인 프로세스

### Phase 4: 포스팅 & 소통

**008-posting**: 자동 실행으로 시간 절약
- 페이스북 API → 인스타그램 자동 포스팅
- 스케줄 자동 실행
- 100개 계정 분산 관리

**009-engagement**: 24시간 팔로워 관리
- DM 자동 응답 (캐릭터 톤 유지)
- 댓글 자동 답글
- 상품 문의 시 구매 링크 전송

### Phase 5: 성과 분석

**010-analytics**: 투명한 성과 확인
- 캠페인별 도달률, 참여율
- 캐릭터별 성과 비교
- 광고주 리포트 자동 생성

## 🔑 핵심 워크플로우

```
1. 페르소나 100명 생성
   ↓
2. 미드저니로 프로필 이미지 제작 (수동)
   ↓
3. 레퍼런스 인스타 계정 크롤링
   ↓
4. 관리자가 크롤링 결과에서 요소 선택 조합
   (이 사진 장소 + 저 사진 제품 + 다른 사진 포즈)
   ↓
5. 나노바나나 프롬프트 자동 생성 → 이미지 생성
   ↓
6. 이미지 맥락 + 레퍼런스 톤으로 캡션 자동 생성
   ↓
7. 승인 후 자동 포스팅 (페이스북 API)
   ↓
8. DM/댓글 자동 응답
```

## 🎯 AI 역할 vs 관리자 역할

**AI가 하는 것**:
- ✅ 캡션 생성 (레퍼런스 톤앤매너 학습)
- ✅ 나노바나나 프롬프트 생성
- ✅ DM/댓글 자동 응답
- ✅ 최적 포스팅 시간 추천
- ✅ 해시태그 자동 추천

**관리자가 하는 것**:
- ✅ 프로필 이미지 제작 (미드저니)
- ✅ 크롤링 결과 조합 (장소+제품+포즈 선택)
- ✅ 최종 승인
- ✅ 광고주 매칭
