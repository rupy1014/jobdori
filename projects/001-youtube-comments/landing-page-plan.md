## 3. 타겟 페르소나

### Primary: 성장 중인 1인 유튜버
- 구독자 1,000~50,000명
- 댓글이 하루 10~100개 달리는 수준
- 영상 제작에 집중하고 싶지만 팬 소통도 중요
- 비용에 민감 (유료 서비스 부담)

### Secondary: 부업 유튜버
- 본업이 있어서 시간 부족
- 효율적인 채널 관리 필요
- 자동화 도구에 관심

## 5. 기술 구현

### 구현 방법
1. **정적 HTML** (현재 프로젝트에 `/public/index.html`로 추가)
2. 또는 **Hono 라우트** (`src/views/landing.ts`)

### 필요한 페이지
- `/` - 랜딩 페이지 (비로그인 사용자)
- `/login` - 로그인/회원가입 (기존)
- `/channels` - 채널 관리 (로그인 후)

### 리다이렉트 로직
```
비로그인 → / (랜딩)
로그인 완료 → /channels (채널 목록)
```
