## 4편 → 5편 연결 포인트

| 4편 | 5편 |
|-----|-----|
| 내 채널 하나 | 멀티 채널 지원 |
| 내 API 키 하드코딩 | 유저별 API 키 |
| 나만 씀 | 회원가입 시스템 |
| 터미널로 배포 | 랜딩 페이지 공개 |
| Cron 자동화 | 스케줄 ON/OFF 설정 |

## 핵심 기술 변화 (4편 vs 5편)

### 4편: 개인용 봇
```typescript
// 하드코딩된 설정
const YOUTUBE_CHANNEL_ID = "내_채널_ID"
const OPENROUTER_API_KEY = process.env.OPENROUTER_API_KEY
```

### 5편: SaaS
```typescript
// 유저별 동적 설정
interface User {
  id: string
  email: string
  openrouterApiKey?: string  // 각자 입력
}

interface Channel {
  userId: string  // 소유자
  youtubeCredentials: {...}  // OAuth로 연결
  settings: {...}  // 채널별 설정
}
```
