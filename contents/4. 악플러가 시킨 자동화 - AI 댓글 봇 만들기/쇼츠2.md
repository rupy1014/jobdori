## 기술 스택 (설명란용)

```
- Runtime: Cloudflare Workers (무료 티어)
- Framework: Hono (경량 웹 프레임워크)
- Storage: Cloudflare KV (키-값 저장소)
- AI 분류: OpenRouter → GPT-4o-mini
- AI 응답: OpenRouter → Gemini 2.0 Flash
- YouTube: YouTube Data API v3 + OAuth 2.0
- 배포: wrangler deploy (CLI 한 줄)
- Cron: 매일 09:00 KST 자동 실행
```

## 핵심 기술 포인트 (대본에 녹아있음)

### 1. 무료 서버 (Cloudflare Workers)
```
"서버비 0원" - Workers 무료 티어로 24시간 운영
"Cron 돌려봤는데" - Workers Cron Triggers
```

### 2. AI 분류 시스템
```
"GPT-4o-mini가 딱 분류해" - 저렴한 모델로 분류
"6가지로 딱 나눠줘" - positive, negative, question, suggestion, reaction, other
```

### 3. 애티튜드 시스템
```
"분류별로 말투가 딱 설정돼있지" - attitudeMap 설정
"악플이면 '품위있게'" - graceful 애티튜드
"질문이면 '전문가처럼'" - expert 애티튜드
```

### 4. 실제 구현 흐름
```
1. YouTube API로 댓글 가져오기 (fetchComments)
2. GPT-4o-mini로 분류 (classifyComment)
3. 분류별 애티튜드 적용
4. Gemini로 응답 생성 (generateRepliesForComments)
5. YouTube API로 답글 게시 (postReply)
```
