# YouTube Shorts Creator - 바이브코딩/자동화 특화

> **"어려운 자동화를 쉽고 재밌게"** - 전문 구현 + 유머 쇼츠 대본 생성기

## 언제 이 스킬을 사용하나요?

- 사용자가 "쇼츠 대본", "1분 영상", "숏폼" 요청할 때
- 바이브코딩, Claude Code, 자동화 주제를 쇼츠로 만들 때
- 기존 긴 대본을 쇼츠로 압축할 때

---

## 핵심 철학

### 두 마리 토끼
```
1. 전문성 - 구현 과정/기술 용어가 들어가야 "와 진짜네" 신뢰감
2. 재미 - 말투가 가벼워야 "끝까지 봐야지" 리텐션
```

### 자동화 쇼츠의 공식
```
[만든 것] + [기술 스택] + [Before/After 임팩트]

예시:
"유튜브 댓글 자동 응답기를 만들었습니다. 코딩 0줄이고요."
"블로그 자동 발행 시스템을 자동화했습니다. Claude + Notion API."
"엑셀 정리를 자동화했습니다. 3시간 → 30초."
```

## 1분 쇼츠 박자 구조

```
[0:00-0:05] 도입 - OO 자동화를 만들었습니다 + 임팩트 수치
[0:05-0:10] 문제 - OO 하다가 지쳐서/손목 나가겠길래
[0:10-0:18] 뜬금 삽입 - 원래 자동화라는 게~ 근데 이건 쉽다
[0:18-0:35] 구현 설명 - 기술 용어 + "딱 돼" 리듬
[0:35-0:45] 작업 과정 - 실제 구현 단계 빠르게 나열
[0:45-0:52] 시청자 말걸기 - 넌 어때요? / 나만 그래?
[0:52-1:00] 결과 - 나이스~ + 임팩트 수치 반복 or 반전
```

## 도입부 템플릿 (자동화 특화)

### 구조
```
[자동화 대상]을/를 [자동화 방법]으로 만들었습니다. [임팩트 수치].
```

### 예시
```
유튜브 댓글 자동 응답기를 만들었습니다. 코딩 0줄이고요.
인스타 DM 자동 응답을 자동화했습니다. 하루 3시간 절약.
블로그 글 발행을 자동화했습니다. Claude + Notion API.
엑셀 데이터 정리를 자동화했습니다. 매크로? 안 씀.
고객 문의 분류를 자동화했습니다. GPT-4 + 구글시트.
커스텀 커맨드로 반복 작업을 자동화했습니다. 10분 → 10초.
```

## 구현 설명 파트 (전문성 + 재미)

### 핵심: 기술 용어는 쓰되, 말투는 가볍게

**X 너무 교육적:**
```
"YouTube Data API v3를 사용하여 댓글을 가져온 후,
OpenAI API를 통해 응답을 생성하고,
다시 YouTube API로 답글을 게시합니다."
```

**O 전문적인데 재밌게:**
```
"댓글 봇이 어떻게 작동하는지 알려면
YouTube Data API에다가 내 채널 연결해서 돌리면 되는데.

봐봐 댓글 들어오면 AI가 딱 읽어. 이렇게.
그럼 이제 성격 파일 열어보면, 말투가 딱 설정돼있지.

'친근하게, 이모지 적당히, 홍보 자연스럽게'
이러면 AI가 내 말투로 답글 단다는 걸 알았지롱."
```

### 기술 용어 삽입 패턴

```
[기술 용어]에다가 [동작] 하면 되는데.
봐봐 [결과]가 딱 [상태]. 이렇게.
그럼 이제 [다음 단계] 열어보면, [결과]가 딱 [상태]지.
```

### 자주 쓰는 기술 용어 (자연스럽게)

| 기술 | 재밌게 표현 |
|------|------------|
| API 연결 | "API에다가 내 계정 연결해서 돌리면" |
| 환경변수 | "API 키 .env에 박아주고" |
| 크론잡/스케줄러 | "1시간마다 자동으로 돌게 해놓으면" |
| 웹훅 | "이벤트 발생하면 알아서 트리거 때려주고" |
| 토큰 | "토큰 넣어주면 인증 끝" |
| JSON | "JSON으로 딱 뱉어주니까" |
| 파싱 | "데이터 파싱해서 필요한 것만 쏙 빼고" |
| 배포 | "서버에 올려놓으면 24시간 자동" |

## 작업 과정 파트 (빠른 나열)

### 구조
```
[목표]하려고
[단계1] 해주고,
[단계2] 넣어주고,
[단계3] 설정해주고,
```

### 예시: 댓글 봇
```
댓글 봇 만들려고
Claude한테 "유튜브 댓글 봇 만들어줘" 때려주고,
YouTube API 키 .env에 박아주고,
성격 파일 personality.md 만들어주고,
"1시간마다 새 댓글 확인" 스케줄러 걸어주고,
```

### 예시: 블로그 자동화
```
블로그 자동 발행 만들려고
Notion API 연결해주고,
Claude한테 "글 써줘" 시키고,
Notion 데이터베이스에 자동 저장되게 해주고,
발행 상태 'Published'로 바꿔주면,
```

### 예시: 엑셀 자동화
```
엑셀 정리 자동화하려고
구글시트 API 연결해주고,
데이터 JSON으로 뽑아주고,
Claude한테 "분류해줘" 시키고,
결과 다시 시트에 박아주면,
```

## 말투 변환 가이드

### Before → After 변환표

| 교육적 스타일 (X) | 자동화 스타일 (O) |
|------------------|------------------|
| "~를 구현했습니다" | "~를 자동화했습니다" |
| "API를 연동합니다" | "API에다가 연결해서 돌리면" |
| "환경변수를 설정합니다" | "API 키 .env에 박아주고" |
| "스케줄러를 설정합니다" | "1시간마다 자동으로 돌게" |
| "결과가 반환됩니다" | "딱 뱉어줘/딱 나와" |
| "완료되었습니다" | "나이스~/뭐야 이거?" |

### 문장 리듬 규칙

```
1. 짧게 끊어친다
   X: "YouTube Data API를 사용하여 댓글을 가져옵니다"
   O: "YouTube API 연결하면. 댓글? 딱 가져와."

2. 기술 용어 + 딱/쏙/팍 조합
   "API 연결하면 딱 돼"
   "JSON으로 쏙 빠져"
   "토큰 박으면 팍 뚫려"

3. 반말+존댓말 혼용
   "~하거든? 근데 API 연결은 좀 무서운데, 넌 어때요?"

4. 자기비하로 마무리
   "나이스~ 더빙만 하면 피곤해 죽겠어"
   "이제 손목은 살았는데 눈이 아파"
```

## 재미 요소 체크리스트

- [ ] **뜬금 삽입** - "원래 자동화라는 게 참 거창하거든? 근데 이건~"
- [ ] **시청자 질문** - "넌 어때요?", "나만 이거 신기해?"
- [ ] **자기비하** - "피곤해 죽겠어", "손목 나가겠어", "눈 빠지겠어"
- [ ] **말장난** - "알았지롱", "때려주고", "박아주고"
- [ ] **과장 수치** - "300개 댓글", "3시간 → 30초", "코딩 0줄"
- [ ] **반전 엔딩** - "뭐야 이거?", "근데 악플은 어떡해?"

## 긴 대본 → 쇼츠 압축 방법

### Step 1: 자동화 포인트 1개만 뽑기
```
10분 대본에서 "이거 자동화했다" 1개 선택
예: "댓글 응답 자동화", "블로그 발행 자동화"
```

### Step 2: 구현 과정 3단계로 압축
```
1. 뭘 연결했나 (API, 서비스)
2. 뭘 설정했나 (설정 파일, 환경변수)
3. 뭘 트리거했나 (실행, 스케줄)
```

### Step 3: 재미 요소 3개 삽입
```
- 뜬금 삽입 1개 (원래 자동화라는 게~)
- 시청자 질문 1개 (넌 어때요?)
- 반전/자기비하 1개 (피곤해 죽겠어)
```

## 자동화 주제 아이디어

### 초급 (진입장벽 낮음)
```
- 커스텀 커맨드로 반복 입력 자동화
- Claude로 이메일 답장 자동 생성
- 노션 템플릿 자동 복제
```

### 중급 (API 연결 필요)
```
- 유튜브 댓글 자동 응답
- 인스타 DM 자동 응답
- 블로그 글 자동 발행 (Notion → Blog)
- 구글시트 데이터 자동 정리
```

### 고급 (풀스택 자동화)
```
- 콘텐츠 수집 → 가공 → 발행 파이프라인
- 고객 문의 자동 분류 + 응답
- SNS 다채널 동시 발행
- 데이터 수집 → 리포트 자동 생성
```

## 레퍼런스 분석 (수리왕 스타일)

### 핵심 패턴
```
1. 도입: "OO을/를 OO했습니다. [임팩트]."
2. 문제: "OO 하다가 지쳐서/나가겠길래"
3. 뜬금: "원래 OO라는 게 참 OO하거든? 근데 이건~"
4. 기술: "[기술]에다가 연결해서 돌리면"
5. 발견: "딱 [결과]. 이렇게. ~라는 걸 알았지롱."
6. 과정: "[동작] 해주고, [동작] 넣어주고, [동작] 설정해주고"
7. 질문: "아 나는 이거 좀 [감정]인데, 넌 어때요?"
8. 결과: "나이스~ [임팩트]? 뭐야 이거?"
9. 반전: "근데 [예외]는 어떡해? 피곤해 죽겠어."
```

### 바이브코딩 적용
```
수리왕: "고장난 GPU를 수리했습니다"
바이브: "댓글 응답을 자동화했습니다"

수리왕: "매츠모즈에다가 명령어 때려놓고"

수리왕: "C0 메모리에 오류가 발생해"

수리왕: "파괴 500도로 조져주면서"
바이브: "Claude한테 '만들어줘' 때려주고"
```
